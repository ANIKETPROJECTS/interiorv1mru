# SJ Interior Studio

## Overview
An interior design website for SJ Interior Studio, built with Create React App (React 19). The site showcases design services, portfolio, testimonials, and contact information.

## Project Structure
- `/src` - React source code
  - `/components` - React components
  - `/styles` - CSS styles
  - `/assets` - Images and assets
- `/public` - Static assets served directly

## Development
- **Framework**: React 19 with Create React App
- **Port**: 5000 (development server)
- **Start command**: `npm start`

## Deployment
- Static site deployment using `npm run build`
- Build output directory: `build`
